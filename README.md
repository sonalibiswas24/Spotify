## Spotify Revenue & User Growth Analysis
# 📊 Project Overview
This project provides a deep dive analysis of Spotify's financial performance and user base growth from 2020 to 2023. By leveraging SQL for data structuring and Tableau for visual storytelling, the project uncovers trends in subscription revenue, advertising impact, and global user acquisition (MAUs).

# 🎯 Key Objectives
Revenue Streams: Evaluate the balance between Premium subscription revenue and Ad-supported revenue.

User Growth: Track the trajectory of Monthly Active Users (MAUs) across different membership tiers.

Profitability Analysis: Compare Gross Profit against operating expenses (R&D, Sales & Marketing, Admin).

Efficiency Metrics: Monitor Average Revenue Per User (ARPU) trends over time.

# 📁 Data Description
The analysis is based on a comprehensive dataset containing quarterly financial reports:

Financials: Total Revenue, Cost of Revenue, Gross Profit, and segment-specific (Premium vs. Ad) financials.

User Metrics: Total MAUs, Premium MAUs, and Ad-supported MAUs.

Operational Spend: R&D costs, Sales and Marketing expenses, and General/Administrative costs.

KPIs: Premium ARPU (Average Revenue Per User).

# 🛠️ Tech Stack
Data Cleaning & Storage: Tableau & xlsx (Data type formatting, calculated fields).

Visualization: Tableau (Interactive Dashboards).

Documentation: GitHub.

# 📈 Visual Insights (Tableau)
The dashboard provides several high-level views:

Revenue vs. Cost: A dual-axis analysis of top-line growth against the cost of maintaining the service.

User Mix: A breakdown of how the ratio of Premium to Free users has shifted over three years.

Operational Efficiency: Visualizing how much Spotify invests in R&D compared to Marketing to drive growth.

Profitability Trends: Identifying quarters with significant margin expansion or contraction.
